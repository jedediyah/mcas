This page contains MCAS data for Mathematics at Nantucket Public Schools. The data was retrieved from the Mass DOE website http://profiles.doe.mass.edu/mcas/ using the automated scripts at https://github.com/jedediyah/mcas and is meant to give a quick picture.

Note that some data points are unavailable during years that a class may not have participated in the MCAS assessment.

These are just snapshots, so use caution when interpreting data to avoid forming strong conclusions.

Please email williamsj@npsk.org if you find any errors or have questions about the graphs.
